DJangoHotel_Python
==================

基于Django的python信息管理系统，用于酒店预订管理

#环境

IDE:JetBrains PyCharm 3.4.1

Python: 2.7.8

Django:1.7

MySQL:5.6.20

------

**Demo:** <http://djangohotel.ga/>

#注意事项
1. 确保已安装python-mysql
1. 在mysql创建djangohotle表
2. 执行python manage.py syncdb创建数据库
3. 并在djangohotel.djangohotel_hotel中创建name为DJango Hotel的数据
